# Emily - B2B Fashion Rental Platform

## What is Emily?
Emily is a B2B fashion rental platform designed for businesses to rent inventory from suppliers with flexible credit terms. Built with multi-tenant architecture, it enables white-label solutions for different brands while maintaining real-time inventory tracking and a trust-based rating system.

## Features
- Multi-tenant architecture
- White-label customization
- B2B credit terms
- Trust/rating system
- Real-time inventory

## Quick Start

```bash
npm install
npm run dev
```
